  1. Enter a Query: Type a natural language query or click an example
  2. API Call: The query is sent to the Lex Machina A2A Agent API
  3. Data Transform: The API response is transformed into chart-compatible format
  4. Visualization: The web component renders the data as a box plot or timeline

Integration Code:

// Query the Lex Machina Agent
async function queryAgent(query) {
    const response = await fetch('http://localhost:10011/api/search', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ query })
    });
    return await response.json();
}
