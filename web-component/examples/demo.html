This repository now includes a framework-agnostic web component for displaying box plots and timeline charts using D3.js.

Features:

  • Box Plot Charts: Display statistical distributions with quartiles, median, and range
  • Timeline Charts: Show temporal trends with smooth curves and interactive tooltips
  • Framework Agnostic: Works with React, Angular, Vue, or plain HTML
  • Shadow DOM: Fully encapsulated styles with no CSS conflicts
  • D3.js Powered: Professional visualizations using D3.js library

Component Structure:

  • web-component/src/lexmachina-chart.js - Main component (12KB)
  • web-component/examples/ - Three working examples
  • web-component/tests/ - Browser-based test suite
  • web-component/README.md - Comprehensive documentation

Usage Example:

<!-- Include D3.js -->
<script src="https://d3js.org/d3.v7.min.js"></script>

<!-- Include the component -->
<script src="web-component/src/lexmachina-chart.js"></script>

<!-- Use the component -->
<lexmachina-chart 
    type="boxplot"
    title="Case Duration by Court"
    width="800"
    height="500">
</lexmachina-chart>

<script>
    // Set data programmatically
    const chart = document.querySelector('lexmachina-chart');
    chart.setData([
        {
            label: 'SDNY',
            min: 30,
            q1: 120,
            median: 180,
            q3: 270,
            max: 450
        }
    ]);
</script>

Data Formats:

Box Plot: Requires min, q1, median, q3, max values for each category

Timeline: Requires date and value pairs for temporal data
